# CKA curriculum practice notes

This question set contains original practice tasks for the local CKA Virtual Lab simulator. The tasks are based on the public CKA curriculum domains and the weights supplied for the 2026-oriented set:

- Storage: 10%
- Troubleshooting: 30%
- Workloads & Scheduling: 15%
- Cluster Architecture, Installation & Configuration: 25%
- Services & Networking: 20%

The 17 tasks are distributed approximately by those weights: 2 Storage, 5 Troubleshooting, 3 Workloads & Scheduling, 4 Cluster Architecture, Installation & Configuration, and 3 Services & Networking.

These are not real exam questions and do not copy real exam items. They are simulator-friendly, original exercises designed to practice common Kubernetes administration skills using only the commands and grading paths supported by this repository.
